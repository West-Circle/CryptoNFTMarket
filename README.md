# 安装与编译

git clone https://github.com/West-Circle/CryptoNFTMarket.git

cd ../CryptoNFTMarket

npm install

truffle compile

truffle migrate

npm start

打开Ganache软件，设置Hostname : 127.0.0.1, Port : 8545, Network ID : 5777

打开Metamask连接，设置网络localhost:8545
